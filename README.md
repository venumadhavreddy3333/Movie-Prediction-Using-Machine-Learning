# Movie Prediction Using Machine Learning

## Overview
This project is a content-based movie recommendation system developed using Machine Learning.

The system recommends movies similar to the user's favourite movie using movie metadata such as genres, cast, director, keywords and tagline.

## Features

- Content-Based Filtering
- TF-IDF Vectorization
- Cosine Similarity Algorithm
- Top 20 Movie Recommendations
- Movie Similarity Search

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

## Algorithm

1. Load movie dataset
2. Preprocess movie features
3. Combine important features
4. Convert text into vectors using TF-IDF
5. Calculate cosine similarity
6. Recommend similar movies
